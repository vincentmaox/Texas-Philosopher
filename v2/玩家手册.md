# 🎮 德州哲学家 v2 · 玩家手册

> v2 版本是 Vite + TypeScript + Canvas 重构，AAA 视觉，16 型 MBTI AI，可打包为 .exe 桌面应用。
> v1 版本（纯前端零依赖）仍在 `web_app/`，使用方式见 `web_app/玩家指南.md`。

---

## 一、三种启动方式

### 方式 1：浏览器（开发版，推荐尝鲜）

```bash
cd v2
npm install        # 首次
npm run dev        # → 浏览器自动开 http://localhost:3000
```

### 方式 2：浏览器（静态部署，发布给朋友）

```bash
cd v2
npm run build      # → dist/ 目录
# 把 dist/ 整个目录传到任意静态服务器或 GitHub Pages
```

### 方式 3：Windows .exe 桌面版（最沉浸）

详见 `v2/TAURI_BUILD.md`：

```bash
# 一次性环境
winget install Rustlang.Rustup
# Visual Studio Installer → "Desktop development with C++"

cd v2
npx @tauri-apps/cli icon 你的图标-1024.png    # 生成图标
npm run tauri:build                            # 出 .exe
# 产物在 src-tauri/target/release/bundle/nsis/*.exe（约 6 MB）
```

## 二、首次进入会发生什么

### 自动弹出 5 步教学

进入牌桌后会弹出 Balatro 风新手教程：

| 步骤 | 内容 |
|---|---|
| 1/5 | 欢迎 — 介绍游戏目标 |
| 2/5 | 底牌 — 你只能看到自己的两张牌 |
| 3/5 | 行动栏 — 弃牌/过牌/跟注/加注/全下 |
| 4/5 | 公共牌 — flop/turn/river 五张共享牌 |
| 5/5 | 学习模式 — EV 反馈和报告卡 |

最后点 "🎲 开始玩吧！" 关闭。系统会记忆已完成（`localStorage.tp_v2_tutorial_done`），下次不再弹。

**想重看？** 设置面板里有 "重置教学" 按钮。

## 四、操作

### 行动按钮

| 按钮 | 中文 | 英文 | 何时出现 |
|---|---|---|---|
| 弃牌 | FOLD | 总是 |
| 过牌 | CHECK | 当前无需跟注时 |
| 跟注 | CALL $X | 当前有人下注时（显示金额） |
| 加注 | RAISE | 你有筹码可加时 |
| 全下 | ALL-IN $X | 总是（显示你剩余筹码） |

### 加注 slider

点 **加注** 会弹出滑条 + 4 个预设：

| 预设 | 含义 |
|---|---|
| Min | 最小加注（当前下注 + 最小加注增量） |
| 1/2池 | 底池一半 |
| 2/3池 | 底池三分之二 |
| 满池 | 等于底池 |

确认后才执行。**取消** 不会扣筹码。

### AI 思考时间

每个 AI 行动前会显示 1.2 秒 "X 正在思考..." 的脉冲指示器。这是给你**判读对手习惯**的窗口：
- 思考时间统一不代表特定意图（避免成为 tell）
- 真正的 tell 在对方的下注**模式**和**台词风格**里

## 六、牌局结束流程

### 一手结束（摊牌或对手全弃牌）

居中弹出大弹窗：

```
        🏆 你赢了
        高对 K
        底池 $240
        赢家：你
    ─────────────────
    INTJ: A♠ K♥ — 一对 A
    你: K♠ K♦ — 三条 K ✓
    ─────────────────
    4 秒后自动开始下一手
```

3.6 秒后弹窗淡出，4 秒后自动发下一手。

### Run 结束（破产或被淘汰）

- 显示完整诊断报告卡（漏洞 + 闪光点 + 总 EV 差）
- 可以选择"重试"立刻开新 Run，元解锁保留

## 八、DeepSeek V4 增强（可选）

不配置也能完整游玩——规则引擎覆盖所有 16 型 AI 决策和台词。

配置方法：设置面板 → "LLM 配置"
- API Base URL：DeepSeek 官方或代理
- API Key：你的 key
- 模型：`deepseek-chat` 或 `deepseek-reasoner`

开启后：
- AI 台词更生动，针对你的具体行为响应
- 显示 AI 的 thinking 过程（reasoner 模式）
- 风格更贴合 16 型人格设定

不会因 LLM 失败而卡住——降级到规则引擎完全无感。

## 九、数据持久化

所有数据存在浏览器 **IndexedDB**（不是 localStorage）：
- `tp_v2_profile` — 玩家档案（连胜、联赛积分、元解锁、设置）
- `tp_v2_current_run` — 当前 Run 的爬塔进度（节点、工具、对手）
- `tp_v2_llm_config` — DeepSeek API 配置（不上传不同步）

**清空数据**：浏览器 F12 → Application → IndexedDB → 删除 `TexasPhilosopherDB`。

**.exe 桌面版**：数据存在系统 WebView2 的数据目录，卸载时不自动清除。

## 十一、常见问题

### Q: 教程关掉了怎么再看？
A: 设置面板有"重置教学"按钮，或者浏览器 F12 → Application → Local Storage → 删除 `tp_v2_tutorial_done`。

### Q: 河牌过牌后没反应？
A: 这个 bug 在 2026-06-09 已修复（commit 56ff15c）。如还遇到，确认你刷新了页面拿到最新代码。

### Q: AI 不说话？
A: 不配置 DeepSeek 也会用预设台词。如果完全空白，看浏览器控制台是否有 JS 错误。

### Q: .exe 启动白屏？
A: 检查 `tauri.conf.json` 的 `devUrl` 与 Vite 端口一致，且系统装了 WebView2（Win11 内置，Win10 需 `winget install Microsoft.EdgeWebView2Runtime`）。

### Q: 牌桌太小或太大？
A: 浏览器版可自由缩放窗口，Canvas 自适应。桌面版固定 1280×800（在 `tauri.conf.json` 改）。
