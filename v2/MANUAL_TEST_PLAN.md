# 德州哲学家 v2 · 人工测试方案

> 完整的端到端 QA 检查清单。所有 P0/P1 项目必须通过才能发布。

---

## 预设环境

- **浏览器**：Chrome 120+ / Edge 120+（首选），Firefox 121+（兼容）
- **分辨率**：1920×1080（桌面）、1280×800（笔记本）、375×812（移动）
- **网络**：可访问 `api.deepseek.com`（LLM 测试需要）
- **数据**：每个测试场景前先在 Console 执行 `indexedDB.deleteDatabase('texas_philosopher_v2'); location.reload()` 重置

## 验收标准

### 最小可发布 (Alpha)

- 所有 **P0** 项目通过
- E2E 流程：地图 → 牌桌 → 报告 → 返回地图 → 刷新保留 ✓
- 至少 1 个完整 Run（5 层）可走通
- 0 console error，无明显视觉问题

### 公开 Beta

- 所有 **P0 + P1** 项目通过
- DeepSeek LLM 集成验证
- 响应式覆盖桌面 + 平板
- Performance > 50ms 单手 EV

### 正式发布

- 所有 **P0 + P1 + P2** 项目通过
- 移动端完整可玩
- PWA Service Worker 上线
- Lighthouse 85+
