# 🎮 德州哲学家 · 玩家日常使用指南

---

## 一、启动游戏

### 方式 1：双击启动（推荐）
找到 `web_app` 文件夹，**双击 `启动游戏.bat`**，游戏自动在默认浏览器打开。

### 方式 2：直接打开
双击 `web_app/index.html`，浏览器直接加载。

> 无需安装 Python / Node.js / 任何依赖。只要有一台能打开浏览器的电脑。

---

## 二、每天怎么玩（10-15 分钟）

### Day 1：出航

1. 打开游戏 → 看到欢迎弹窗 → 点 **"出航 →"**
2. 进入 **探宝航海图** → 6 片海域呈现在你面前
3. 先点 **训练码头**（免费，零风险）熟悉手感
4. 打几手牌，观察 AI 对手的风格
5. 退出后选择 **近海浅滩**（买入 50）开始正式训练

### 每天：日常流程

```
打开游戏
  → 看到每日登录奖励弹窗（连胜 ×5 思维评分）
  → 进入"每日"页面，查看今日 3 个挑战
  → 回到地图，选一个鱼塘打牌
  → 每手牌做决策后，留意右上角的评分 Toast
  → 打 10-15 分钟，时间池用完就停
  → 关闭浏览器，进度自动保存
```

## 三、界面导航

- ⏳ **时间池** — 每日恢复，用完今天就结束
- 🧠 **思维评分** — 你的真实排名
- 🔥 **连胜** — 连续登录天数，断一天归零

## 四、打牌操作

### 鼠标操作

当你轮到行动时，底部会出现操作栏：

| 按钮 | 作用 |
|---|---|
| 弃牌 (F) | 扔掉手牌，放弃本手 |
| 过牌 (C) | 不下注，让给下一位（无人下注时） |
| 跟注 (C) | 跟上当前下注额（有人下注时） |
| 加注 (R) | 滑块/输入框设金额，点加注 |
| All-in (A) | 把所有筹码推入 |

### 键盘快捷键

| 键 | 操作 |
|---|---|
| **F** | 弃牌 |
| **C** | 跟注 / 过牌 |
| **R** | 加注（当前输入框金额） |
| **A** | All-in |

> 输入框获得焦点时快捷键不触发，避免误操作。

### 加注操作

1. 拖动滑块 或 直接在输入框输入金额
2. 点击 **加注 (R)** 确认
3. 金额 = 你本次额外投入的筹码（已含跟注部分）

## 五、决策评分 — 游戏的灵魂

**每次你做行动，系统立即给反馈。** 这不是输赢评分，是思维评分：

### 好决策（加分）

| 场景 | 评分 | 含义 |
|---|---|---|
| 输了大锅后冷静弃烂牌 | +8 | 独立事件思维 |
| 弃掉 -EV 烂牌 | +5 | 完美弃牌 |
| 强牌价值下注 | +8 | 捡钱模式 |
| pot odds 合理的跟注 | +3 | +EV 决策 |
| 赢钱后仍保持纪律 | +5 | 不被胜利冲昏 |

### 坏决策（扣分）

| 场景 | 评分 | 含义 |
|---|---|---|
| 输大锅后 All-in 翻本 | -15 | 心理账户污染 |
| -EV 跟注大额 | -12 | 为别人期望值买单 |
| Preflop All-in 烂牌 | -20 | 梭哈反面教材 |
| 强牌不过牌 | -5 | 错失价值 |
| 赢了后飘了乱加注 | -10 | 过度自信 |

### 中性

大部分决策是中性的（+0），系统不会每手都给你压力。

## 六、鱼塘选择策略

### 看密度

每个鱼塘节点下方有密度条：

- 🟢 绿色圆点 = 鱼（弱对手，好打）
- 🔴 红色圆点 = 鲨鱼（强对手，难打）

**鱼多 → 值得深入；鲨多 → 快速离开或跳过。**

### 看突变

鱼塘可能带有橙色突变标签，例如：
- 🌫️ 密度迷雾 — 首手看不到对手是谁
- 👁️ 读心术 — 能看一个对手底牌（福利！）
- ⚓ 沉没成本 — 弃牌有代价

**如果突变对你不利，换一个鱼塘。低成本试错优于高投入梭哈。**

### 推荐路线

| 阶段 | 选择 | 原因 |
|---|---|---|
| 新手 | 训练码头 → 近海浅滩 | 零/低成本，鱼多 |
| 入门 | 鱼群密集湾 | 1 个突变体验极限规则 |
| 进阶 | 混合洋流 | 鱼鲨混杂，考验密度判断 |
| 高手 | 鲨鱼礁 | 危险但收益高 |
| 大师 | 虚空深渊 | 需解锁，3 个突变同时生效 |

## 七、与 AI 对话

### AI 主动说话

AI 在关键时刻会主动对你说话：
- 你 All-in 时
- 你弃牌时
- 你连续弃牌时
- AI 自己 All-in 时
- 有人赢大锅 / 输大锅时

### 你跟 AI 说话

1. 点击右下角 **"对AI说话"** 按钮
2. 在输入框打字，按回车发送
3. AI 会根据角色性格回应

> 无 LLM 时，AI 用预设台词回复。配置 LLM 后，AI 可即兴生成更丰富的对话。

## 八、AI 大模型配置（让 AI 更聪明）

如果你想让 AI 说出更有趣、更贴合角色的话：

1. 点击 **设置** 页
2. 填写：
   - **API Base URL**：你使用的 API 地址
   - **API Key**：你的密钥
   - **模型名**：如 `gpt-4o-mini`、`deepseek-chat` 等
3. 勾选 **启用 AI 对话**
4. 点击 **测试连接** → 显示 "连接成功"
5. 点击 **保存**

常见 API：

| 平台 | Base URL | 推荐模型 |
|---|---|---|
| OpenAI | `https://api.openai.com/v1` | `gpt-4o-mini` |
| DeepSeek | `https://api.deepseek.com/v1` | `deepseek-chat` |
| 火山方舟 | `https://ark.cn-beijing.volces.com/api/v3` | `doubao-pro-32k` |
| 阿里云 | `https://dashscope.aliyuncs.com/compatible-mode/v1` | `qwen-plus` |

> 你的 API Key 只存在浏览器本地，不会上传到任何服务器（除了你配置的 API 端点）。

## 九、每日系统

### 连胜

- 每天打开游戏 = +1 连胜
- 连胜 × 5 = 当日思维评分奖励
- **断一天 = 归零！** 多邻国式压力

### 每日任务

每天随机 3 个任务，例如：
- 弃掉 10 手烂牌
- 探测 3 个新鱼塘
- 连续 5 手不受上局影响
- 连输 3 手后冷静弃牌

完成任务获得 XP 和思维评分。

### 时间池

- 每天 30 分钟（解锁"时间编织者"技能后 +10）
- 每手牌消耗 1 分钟
- 用完今天就不能再打了

## 十、技能树

用 XP（通过好决策积累）解锁被动技能：

| 技能 | 消耗 | 效果 |
|---|---|---|
| 🏃 快速弃牌 | 30 XP | 弃牌不消耗时间池 |
| 🎣 低成本探测 | 50 XP | 每局第一手买入减半 |
| 📡 鱼塘雷达 | 60 XP | 进入前看到对手等级 |
| 👃 乌龙指嗅觉 | 80 XP | 8% 概率发现捡钱机会 |
| 🛡️ 情绪隔离 | 100 XP | 连输后损失减少 40% |
| ⏳ 时间编织者 | 100 XP | 每天时间池 +10 分钟 |
| 🔰 独立事件护盾 | 120 XP | 上手结果不再显示干扰 |
| 👁️ 频域之眼 | 150 XP | 看到对手弃牌/加注频率 |
| 🔮 虚空谐振 | 200 XP | 三频共振时 +15% 手牌强度 |

推荐解锁顺序：快速弃牌 → 低成本探测 → 鱼塘雷达 → 情绪隔离

## 十一、存档管理

### 自动保存
游戏每 30 秒自动保存，关闭浏览器也不丢进度。

### 手动导出
设置页 → **导出存档** → 下载 JSON 文件。可用于备份或换电脑迁移。

### 导入存档
设置页 → **导入存档** → 选择之前导出的 JSON 文件。页面会自动刷新。

### 重置
设置页 → **重置全部进度** → 确认后从零开始。**不可逆！**

## 十二、训练 30 天后的预期效果

这不是承诺，这是训练目标：

- 看到 22 不会因为"昨天输了"而 All-in 翻本
- 看到鱼塘密度不对会立刻换桌，不再"再试一手"
- 听到 "All-in 翻本" 会立刻识别为情绪信号
- 强牌时会坚定价值下注，不再"怕跑鱼"
- 知道什么时候离开牌桌比什么时候参与更重要

**每天 10-15 分钟，坚持 30 天。**
