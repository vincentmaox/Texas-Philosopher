# 德州哲学家 · Texas Philosopher

> 一款以"频域决策 / 独立事件 / 鱼塘密度 / 捡钱思维"为核心训练目标的 AI 驱动德州扑克游戏。
> 参考小丑牌（突变）+ 杀戮尖塔（地图）+ 多邻国（每日激励）。

## 🚀 一秒启动

**双击 `index.html`** —— 在任意浏览器中打开即可，无需安装任何东西。

游戏会自动保存到浏览器 localStorage。

## 🎯 这游戏想训练什么

不是教你"赢德州扑克"。是把以下思维做成你身体的反射：

| 训练目标 | 游戏机制 |
|---|---|
| **独立事件**：上一局不影响下一局 | Scoring 每次行动后实时给 +/- XP |
| **成本管理**：pool 不被耗干 > 赢一把大的 | 资金池 + 时间池双资源，归零即 Game Over |
| **鱼塘密度**：低成本判断牌桌是否值得继续 | 每个鱼塘有密度标签，不值得立刻换桌 |
| **捡钱而非挣钱**：只打正期望值 | "勉强参与"的痛苦远大于"错过机会" |
| **快速试错**：多局探测胜过单局投入 | 探宝航海图，鼓励频繁换桌 |

## 🎮 核心玩法

### 1. 探宝航海图（地图）
- 6 个鱼塘节点，每个有：买入、盲注、密度、突变、对手列表
- 选择进入 → 打几手 → 判断是否值得继续 → 离开换桌
- **港口事件**：随机遭遇（修船/老渔夫情报/哲学考验）
- 高级海域（虚空深渊）需要解锁

### 2. 6 名 AI 对手（每个都会说话！）
- 老渔夫（极致耐心）
- 梭哈狂人（情绪驱动）
- 密度探测者（低成本试错）
- 学院派（GTO）
- 后浪（激进学习）
- **虚空建筑师**（频域思维 · 你的镜像）

每个 AI 在你 All-in / 弃牌 / 连续弃牌等关键时刻会主动对你说话。**你也可以打字跟他们说话**（需开启 LLM）。

### 3. 突变规则（小丑牌风格）
- 沉没成本诅咒、梭哈狂热、信息黑洞、密度迷雾
- 时间加速、独立事件封印、捡钱法则、鲨群进食、读心术 等 12+ 种

### 4. 每日训练（多邻国风格）
- 🔥 连续登录天数 / 中断归零
- 3 个每日挑战（弃 10 手烂牌 / 探测 3 个鱼塘 / 连续 5 手不受上局影响）
- 每日时间池恢复 30 分钟

### 5. 思维技能树（杀戮尖塔风格）
- 快速弃牌 / 鱼塘雷达 / 情绪隔离 / 独立事件护盾 / 频域之眼 / 虚空谐振
- 用思维 XP 解锁

### 6. 即时决策评分
每次你做行动，立即得到反馈 toast：
- `+10 独立事件思维！上局输光没有让你慌乱报复。`
- `-15 心理账户污染！上局输大锅强行翻本。`
- `+8 捡钱模式！好牌就要敢于价值下注。`
- `-20 preflop 梭哈烂牌！文章讽刺的反面教材。`

## 🤖 AI 大模型配置（可选，强烈推荐）

未配置 → AI 用预设台词，游戏完全可玩。
配置后 → AI 会即兴生成符合角色的台词、对你的话做出反应。

进入"设置"页填：
- **API Base URL**：例如 `https://api.openai.com/v1` 或 `https://api.deepseek.com/v1` 或 `https://ark.cn-beijing.volces.com/api/v3`（火山方舟）
- **API Key**：你的密钥
- **模型名**：`gpt-4o-mini` / `deepseek-chat` / `doubao-pro-32k` / `qwen-plus` 等
- 勾选"启用 AI 对话"
- 点"测试连接"确认

所有数据只在浏览器本地存储，不会上传任何服务器（除你自己配置的 AI 端点外）。

## 🌊 设计哲学（开发者说明）

本项目的目标不是让你赢钱，而是让你的**身体反应**和**频域思维**对齐。

> 频域决策：不看 K 线看键位，不赌走势赌频谱，不求预测求采样，不做持仓做套利。
> 非对称风险：下行有限 + 上行凸性 + 高暴露 = 立即行动。

每天 10-15 分钟，连续训练 30 天，你会发现：
- 看到 22 不会再因为"昨天输了"而 All-in 翻本
- 看到一个鱼塘密度不对会立刻换桌而非"再试一手"
- 听到 "All-in 翻本" 会立刻识别为情绪信号
